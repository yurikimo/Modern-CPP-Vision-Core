# 🧠 Modern C++ / Computer Vision / AR-VR Learning Repository

This repository documents the showcase of:

- Modern C++ (C++17+)
- Computer Vision
- Real-time systems
- AR/VR development
- Unreal Engine (C++)

---

## 🎯 Objective

Build a **showcase** focused on:

- Performance-critical systems
- Memory-safe architecture
- Real-time processing pipelines
- Industry-relevant AR/VR technologies

---

## 📂 Repository Structure

This repository is **branch-driven**.

- `main` → Overview
- Feature branches → Individual topics, experiments, and implementations

Each branch contains:
- A focused implementation
- Its own `README.md`
- Technical explanations and usage details

---

## 🧪 Topics Covered

- Memory management (RAII)
- Move semantics
- STL containers
- Multithreading
- Computer vision pipelines
- Real-time processing
- Unreal Engine C++ systems
- AR/VR fundamentals

---

## 🚀 How to Use This Repository

1. Browse branches:
   ```
   git branch -a
   ```

2. Switch to a topic:
   ```
   git checkout <branch-name>
   ```

3. Read the branch-specific README for details
